# Projet-Code

## Exécution

- Se placer à la racine du projet
- Installer les dépendances
    ```bash
    pip install -r requirements.txt
    ```
- Lancer le programme
    ```bash
    python src/main.py
    ```

## Tests

- Se placer à la racine du projet
- Installer les dépendances
    ```bash
    pip install -r requirements.txt
    ```
- Lancer les tests
    ```bash
    python -m unittest discover -s tests
    ```
